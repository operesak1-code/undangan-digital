# 🚀 Quick Start - Undangan Digital

## Mulai Dalam 5 Menit!

### 1. Upload ke GitHub (3 menit)

1. Login ke [GitHub](https://github.com/)
2. Buat repository baru: `undangan-digital`
3. Upload semua file ke repository
4. Aktifkan GitHub Pages di Settings → Pages

### 2. Akses Aplikasi (1 menit)

Buka: `https://username.github.io/undangan-digital/undangan.html`

### 3. Buat Undangan (1 menit)

1. Pilih template
2. Isi nama & tanggal
3. Upload foto
4. Salin link & bagikan!

---

## 📁 File Penting

| File | Fungsi |
|------|--------|
| `undangan.html` | Halaman utama aplikasi |
