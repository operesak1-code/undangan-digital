# Musik Latar untuk Undangan Digital

## Cara Menambahkan Musik

1. **Siapkan file MP3** - Pastikan file musik dalam format `.mp3`
2. **Ukuran file** - Usahakan ukuran file kecil (< 2MB) agar loading cepat
3. **Hak cipta** - Gunakan musik bebas royalti atau musik orisinal Anda

## File Musik yang Tersedia

Saat ini folder ini kosong. Anda perlu menambahkan file musik sendiri.

### Rekomendasi Sumber Musik Gratis

1. **YouTube Audio Library** - https://www.youtube.com/audiolibrary
2. **Bensound** - https://www.bensound.com/
3. **Free Music Archive** - https://freemusicarchive.org/
4. **Incompetech** - https://incompetech.com/music/

### Cara Menambahkan

1. Download musik dari sumber gratis di atas
2. Rename file dengan nama yang deskriptif (contoh: `romantic-piano.mp3`)
3. Upload ke folder ini
4. Update dropdown di `index.html` dengan nama file yang sesuai

## Contoh File Musik

```
templates/music/
├── romantic-piano.mp3      (Piano romantis)
├── acoustic-guitar.mp3     (Gitar akustik)
├── classical-violin.mp3    (Biola klasik)
├── smooth-jazz.mp3         (Jazz lembut)
└── soft-ambient.mp3        (Ambient lembut)
```

## Tips

- Gunakan musik instrumental untuk suasana yang lebih khidmat
- Hindari musik dengan lirik yang terlalu keras
- Test di berbagai browser untuk memastikan kompatibilitas
