# Music Folder - Undangan Digital

## Cara Menambahkan Musik

Folder ini untuk menyimpan file musik/MP3 untuk latar belakang undangan.

### Langkah-langkah:

1. **Siapkan file MP3** (ukuran kecil, maksimal 3-5 MB agar loading cepat)

2. **Tambahkan file ke folder ini** dengan nama yang deskriptif:
   - `romantic-1.mp3`
   - `romantic-2.mp3`
   - `acoustic-1.mp3`
   - `classical-1.mp3`
   - `jazz-1.mp3`

3. **Update dropdown musik** di `undangan.html`:
   ```html
   <option value="music/nama-file.mp3">Nama Lagu</option>
   ```

### Sumber Musik Gratis:

- **YouTube Audio Library** - https://www.youtube.com/audiolibrary
- **Free Music Archive** - https://freemusicarchive.org/
- **Bensound** - https://www.bensound.com/
- **Incompetech** - https://incompetech.com/

### Tips:
- Gunakan musik instrumental untuk suasana lebih khidmat
- Pastikan lisensi musik bebas royalti
- Convert ke MP3 dengan bitrate 128kbps untuk ukuran kecil
- Test di berbagai browser untuk kompatibilitas
