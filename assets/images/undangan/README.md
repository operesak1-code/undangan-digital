# Placeholder Images for Undangan Digital

## Default Images

Aplikasi ini menggunakan gambar dari Unsplash sebagai default. Anda dapat menggantinya dengan:

### Opsi 1: Gunakan URL Eksternal
- Upload foto ke layanan hosting gambar:
  - **Imgur** - https://imgur.com/
  - **Google Photos** (dapatkan link langsung)
  - **Cloudinary** - https://cloudinary.com/
  
### Opsi 2: Simpan Lokal
1. Simpan file gambar di folder `assets/images/undangan/`
2. Gunakan path relatif: `assets/images/undangan/foto-anda.jpg`

### Rekomendasi Ukuran Gambar:
- **Foto Pengantin**: 800x600px atau 4:3 ratio
- **Background**: 1920x1080px atau 16:9 ratio
- **Format**: JPG untuk foto, PNG untuk grafis dengan transparansi
- **Ukuran file**: Maksimal 500KB untuk loading cepat

### Tips Optimasi Gambar:
- Gunakan TinyPNG (https://tinypng.com/) untuk kompresi
- Resize gambar sebelum upload
- Gunakan format WebP untuk browser modern
